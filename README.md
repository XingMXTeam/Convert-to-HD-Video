# Convert to HD Video

> A small video-processing experiment repository.

This repository is a work-in-progress space for testing video-processing scripts and repeatable media-export workflows.

## Purpose

The goal is to document and organize experiments around:

- video format conversion
- output preset management
- local media-processing scripts
- reusable creator-tooling utilities

## Suggested workflow

```text
input video
  -> processing script
  -> export settings
  -> output video
```
